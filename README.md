# 🚗 Agente de Busca de Carros com IA

Sistema inteligente que permite buscar veículos usando linguagem natural.

Exemplo:
"quero um SUV até 2020 por menos de 60 mil"

---

## 🧠 Como funciona

O sistema interpreta o texto do usuário e transforma em filtros:

- Categoria (SUV, Sedan, Hatch…)
- Preço máximo
- Ano
- Combustível
- Câmbio

Depois consulta o banco de dados e retorna os veículos compatíveis.

---

## ⚙️ Tecnologias utilizadas

- FastAPI (API backend)
- Gradio (interface de chat)
- SQLAlchemy (banco de dados)
- SQLite (armazenamento)
- Python
